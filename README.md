# 🍔 Projeto: Pedido da Lanchonete (C#)

## 📌 Descrição
Este projeto é um programa em **C#** que simula o sistema de pedidos de bebidas em uma lanchonete.  
O usuário pode escolher bebidas de um cardápio, selecionar quantas unidades deseja e, no final, visualizar um resumo do pedido com o valor total da compra.  

É um exercício prático para treinar:
- Estruturas de repetição (`do while`)
- Estrutura condicional (`switch`)
- Criação de classes e listas (`List<>`)
- Operações com valores e cálculos automáticos

---

## 🚀 Funcionalidades
✅ Exibe um cardápio de bebidas com preços  
✅ Permite que o usuário escolha **quantidade** de cada bebida  
✅ Calcula automaticamente o subtotal de cada item e o valor total  
✅ Exibe um **resumo final do pedido** antes de encerrar  

---

## 📖 Exemplo de uso

```bash
